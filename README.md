- **Aurora to Starry Nights** — A collection of 19 English poems exploring themes of wonder, emotion, and the night sky
- **Those Late Night Walks And Conversations** — A collection of 3 original plays written in a unique blend of Hindi and English, capturing intimate moments and emotions

Each piece is thoughtfully curated to reflect my creative journey and artistic expression. Whether you're exploring poetry, theatrical works, or short stories, you'll find a diverse range of creative writing here.

## For Developers

### Setup & Installation

```bash
# Clone the repository
git clone https://github.com/sanyogita123shukla/Writing_Portfolio.git
cd Writing_Portfolio

# Install dependencies
npm install

# Run development server
npm run dev
```

Open http://localhost:5173 in your browser.

### Edit Your Content

**`content/site.json`** — Update profile, contact info, and book chapters

**`content/aurora-poems-source.txt`** — Add or edit poems (separated by `---`)

**`src/pages/`** — React components for different pages

**`src/styles/global.css`** — Design and styling

### Add a New Poem

Edit `content/aurora-poems-source.txt` and add:

```
New Poem Title
--------------

Your poem content here...

---
```

### Build for Production

```bash
npm run build
npm run preview
```

The `dist` folder is ready to deploy!
